# Jeu de Go - Projet Programmation Web

Ce projet est une implémentation d'un **Jeu de Go** développée dans le cadre de ma formation. 

##  Fonctionnalités
- Plateau de jeu interactif.
- Pose de pions (Noirs et Blancs).
- Interface web responsive.

##  Technologies utilisées
- **HTML5** pour la structure.
- **CSS3** pour le design (dossier `/css`).
- **JavaScript** pour la logique du jeu,
-  **Bootstrap 5.3** pour le design responsive et les composants (Navbar, etc.).

##  Structure du projet
- `index.html` : Page principale du jeu.
- `css/` : Contient les fichiers de styles.
- `Img/` : Images et ressources graphiques.
